# popc

Simple opencode sandbox:

![Screencast](./screencast.gif)

## Installation

```sh
mkdir -p $HOME/git/popc
cd $HOME/git/popc
git clone git@github.com:mtratsiuk/popc.git

ln -s $HOME/git/popc/popc.sh $HOME/.local/bin/popc
chmod +x $HOME/.local/bin/popc
```

## Usage

```sh
cd ~/git/project
popc
```
